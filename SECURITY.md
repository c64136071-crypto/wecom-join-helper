# Security Policy

## Supported Versions

Security fixes are provided for the latest published release.

## Reporting A Vulnerability

Do not post account data, chat screenshots, company information, logs, or
security-dialog captures in a public issue. Use GitHub's private vulnerability
reporting feature when it is available for this repository.

Include the application version, Windows version, WeCom version, and a minimal
description that contains no personal or company data. The project does not
request passwords, QR codes, login tokens, or remote access.

## Security Boundary

WeCom Join Helper performs visible desktop UI automation for an already logged
in user. It does not automate login or verification prompts, call private APIs,
disable security controls, or interfere with verification safeguards. Users must follow
their organization's policies and use the software only where authorized.
